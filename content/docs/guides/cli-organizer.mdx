This guide walks through the full lifecycle of a hackathon using the Oatmeal CLI — from creation to publishing results.

## Prerequisites

Install the CLI and sign in:

<CodeBlock language="bash" showLineNumbers code={"npm install -g @oatmeal/cli\noatmeal login"}>
  <CodeBlockHeader>
    <CodeBlockFilename>Setup</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

## Step 1: Create a Hackathon

<CodeBlock language="bash" showLineNumbers={false} code={"oatmeal hackathons create --name \"Spring Hack 2026\" --slug spring-hack-2026"}>
  <CodeBlockHeader>
    <CodeBlockFilename>Terminal</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

Save the hackathon ID from the output — you'll need it for subsequent commands. Or use the slug directly:

<CodeBlock language="bash" showLineNumbers={false} code={"oatmeal hackathons get spring-hack-2026"}>
  <CodeBlockHeader>
    <CodeBlockFilename>Verify</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

## Step 2: Define Judging Criteria

Set up how submissions will be scored:

<CodeBlock language="bash" showLineNumbers code={"oatmeal judging criteria create spring-hack-2026 \\\n  --name \"Innovation\" --description \"How novel is the approach?\" \\\n  --max-score 10 --weight 30\n\noatmeal judging criteria create spring-hack-2026 \\\n  --name \"Technical Execution\" --description \"Code quality and architecture\" \\\n  --max-score 10 --weight 40\n\noatmeal judging criteria create spring-hack-2026 \\\n  --name \"Presentation\" --description \"Demo quality and clarity\" \\\n  --max-score 10 --weight 30"}>
  <CodeBlockHeader>
    <CodeBlockFilename>Create criteria</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

Verify your criteria:

  <CodeBlockHeader>
    <CodeBlockFilename>List criteria</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

## Step 4: Set Up Prizes

Create prizes for winners:

<CodeBlock language="bash" showLineNumbers code={"oatmeal prizes create spring-hack-2026 \\\n  --name \"Grand Prize\" --type monetary --value 5000\n\noatmeal prizes create spring-hack-2026 \\\n  --name \"Runner Up\" --type monetary --value 2500\n\noatmeal prizes create spring-hack-2026 \\\n  --name \"Best Design\" --type custom --value \"Design toolkit license\""}>

## Step 5: Assign Judges to Submissions

Once the submission deadline passes, distribute submissions to judges:

<CodeBlock language="bash" showLineNumbers={false} code={"oatmeal judging auto-assign spring-hack-2026 --per-judge 5"}>
  <CodeBlockHeader>
    <CodeBlockFilename>Auto-assign</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

This evenly distributes submissions so each judge reviews approximately 5 projects. Check the assignment breakdown:

<CodeBlock language="bash" showLineNumbers={false} code={"oatmeal judging judges list spring-hack-2026"}>
  <CodeBlockHeader>
    <CodeBlockFilename>Check progress</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

The output shows each judge's completion progress (e.g., "3/5 scored").

## Step 6: Monitor Judging Progress

While judges are scoring, monitor their progress:

<CodeBlock language="bash" showLineNumbers code={"# See overall progress\noatmeal judging judges list spring-hack-2026\n\n# View all assignments\noatmeal judging assignments list spring-hack-2026"}>
  <CodeBlockHeader>
    <CodeBlockFilename>Monitor</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />

## Step 7: Calculate and Publish Results

Once all judges have scored, calculate the final rankings:

<CodeBlock language="bash" showLineNumbers code={"# Calculate rankings from scores\noatmeal results calculate spring-hack-2026\n\n# Review the results before publishing\noatmeal results get spring-hack-2026\n\n# Publish when ready\noatmeal results publish spring-hack-2026"}>

    <CodeBlockFilename>Results workflow</CodeBlockFilename>

## Quick Reference

Here's the complete organizer workflow in one block:
