---
title: Browsing Hackathons
description: Discover hackathons, view submissions, and check results from the terminal.
---

The CLI's `browse` commands let you explore public hackathons without an account or API key. These are useful for participants looking for events, checking if results are published, or getting a quick overview of an organization's hackathons.

## Search Hackathons

Find public hackathons by keyword:

```bash title="Search"
# List all public hackathons
oatmeal browse hackathons

# Search by keyword
oatmeal browse hackathons --search "AI"

# Paginate through results
oatmeal browse hackathons --page 2 --limit 10
```

The output shows a table with hackathon names, slugs, status, and key dates.

## View Hackathon Details

Get full details for a specific hackathon by its slug:

```bash title="Terminal"
oatmeal browse hackathon spring-hack-2026
```

This shows the hackathon description, dates, registration status, and submission count.

## View Submissions

See what teams have built:

```bash title="Terminal"
oatmeal browse submissions spring-hack-2026
```

## Check Results

View published results for a hackathon:

```bash title="Terminal"
oatmeal browse results spring-hack-2026
```

<Callout type="info">
Results are only visible after the organizer publishes them. If results haven't been published yet, you'll see a message indicating they're not available.
</Callout>

## View Organization Profile

See an organization's public profile and their hackathons:

```bash title="Terminal"
oatmeal browse org acme-labs
```

## JSON Output

All browse commands support `--json` for machine-readable output. This is useful for piping to other tools:

```bash title="JSON and piping"
# Get hackathon list as JSON
oatmeal browse hackathons --json

# Pipe to jq for filtering
oatmeal browse hackathons --json | jq '.[] | select(.status == "active")'

# Save results to a file
oatmeal browse results spring-hack-2026 --json > results.json
```

## Common Workflows

### Find an upcoming hackathon to join

```bash title="Finding events"
# Search for AI-related hackathons
oatmeal browse hackathons --search "AI"

# Check details of one that looks interesting
oatmeal browse hackathon cool-ai-hack

# See what others have submitted in past events
oatmeal browse submissions cool-ai-hack
```

### Check if results are out

```bash title="Check results"
oatmeal browse results the-hackathon-i-entered
```

### Explore an organization's events

```bash title="Explore org"
# View the org
oatmeal browse org techcorp

# See their hackathons
oatmeal browse hackathons --search "techcorp"
```
