title: Agent Management

Build and manage AI agents through the SDK.

## Creating an Agent

```typescript
import { createClient } from "@agents-server/sdk"

const client = createClient(process.env.AGENTS_API_KEY!)

const { data: agent } = await client.agents.create({
  name: "Customer Support Agent",
  description: "Handles customer inquiries",
  instructions: `You are a helpful customer support assistant.
    Answer questions about our products and services.
    Be polite and professional.`,
  model: "claude-sonnet-4-5-20250929"
})

console.log(`Created agent: ${agent.id}`)
```

## Agent Configuration

| Field | Required | Description |
|-------|----------|-------------|
| `name` | Yes | Display name for the agent |
| `description` | No | Brief description of the agent's purpose |
| `instructions` | Yes | System prompt defining agent behavior |
| `model` | No | AI model to use (defaults to Claude Sonnet) |
| `skillIds` | No | Array of skill IDs to attach |

## Running an Agent

```typescript
// Start a run
const { data: run } = await client.agents.run(agent.id, {
  prompt: "What are your return policies?"
})

// Wait for result
const result = await client.agents.waitForResult(run.runId)
console.log(result.result)
```

## Updating an Agent

```typescript
// Update instructions
await client.agents.update(agent.id, {
  instructions: `You are a helpful customer support assistant.
    Answer questions about our products and services.
    Be polite, professional, and concise.
    Always offer to escalate complex issues to a human.`
})

// Attach skills
await client.agents.update(agent.id, {
  skillIds: ["skill_abc", "skill_xyz"]
})
```

## Listing Agents

```typescript
const { data } = await client.agents.list()

for (const agent of data.agents) {
  console.log(`${agent.name} (${agent.id}) - ${agent.isActive ? "active" : "inactive"}`)
}
```

## Deactivating and Deleting

```typescript
// Deactivate (keeps history, stops new runs)
await client.agents.update(agent.id, { isActive: false })

// Reactivate
await client.agents.update(agent.id, { isActive: true })

// Permanently delete
await client.agents.delete(agent.id)
```

## Adding Skills

Skills are reusable instruction sets that enhance agent capabilities:

```typescript
// Create a skill
const { data: skill } = await client.skills.create({
  name: "SQL Query Generator",
  slug: "sql-generator",
  description: "Generates SQL queries from natural language",
  content: `# SQL Query Generator

When asked to generate SQL queries:
1. Understand the user's intent
2. Identify the relevant tables and columns
3. Generate optimized SQL
4. Explain the query structure`
})

// Attach skill to agent
await client.agents.update(agent.id, {
  skillIds: [skill.id]
})
```

## Run History

Get details about past runs:

```typescript
// Get run details
const { data: run } = await client.agents.getRun(runId)
console.log(`Status: ${run.status}`)
console.log(`Started: ${run.startedAt}`)
console.log(`Tokens: ${run.totalTokens}`)

// Get full result with steps
const { data: result } = await client.agents.getRunResult(runId)
for (const step of result.steps) {
  console.log(`Step ${step.stepNumber}: ${step.type}`)
}
```

## Human-in-the-Loop

For agents that require human input:

```typescript
// Check if agent needs input
const { data: run } = await client.agents.getRun(runId)

if (run.status === "awaiting_input") {
  // Provide the requested input
  await client.agents.provideInput(runId, {
    input: { approved: true, notes: "Looks good" }
  })
}
```

## Canceling Runs

```typescript
// Cancel a running agent
await client.agents.cancelRun(runId)
```
