Every hackathon in Oatmeal belongs to an organization. This guide covers setting up your organization profile, which appears on public event pages and your dedicated organization page at `/o/{slug}`.

## Overview

Your organization profile includes:
- **Name and slug** — Displayed on event pages and in the `/o/{slug}` URL
- **Description** — Shown on your public organization page
- **Website URL** — Link to your website
- **Logos** — Light and dark mode variants for proper theming

## Setting Up Your Profile

Update your organization details via the API:

```bash title="Update organization profile"
curl -s -X PATCH "$BASE_URL/api/dashboard/org-profile" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "AGI Ventures Canada",
    "slug": "agi-ventures",
    "description": "Building the future of hackathons in Canada",
    "websiteUrl": "https://agiventures.ca"
  }' | jq .
```

## Choosing a Slug

The slug determines your public URL (`/o/{slug}`) and appears in various places across the platform. Slugs must be:
- At least 3 characters
- Lowercase letters, numbers, and hyphens only
- Unique across all organizations

Check slug availability from the dashboard at **Settings > Organization**. The availability check requires a browser session and is not accessible via API key.

## Uploading Logos

Upload separate logos for light and dark themes to ensure proper display across user preferences:

```bash title="Upload light logo"
curl -s -X POST "$BASE_URL/api/dashboard/upload-logo" \
  -H "Authorization: Bearer $API_KEY" \
  -F "file=@logo-light.png" \
  -F "variant=light" | jq .
```

```bash title="Upload dark logo"
curl -s -X POST "$BASE_URL/api/dashboard/upload-logo" \
  -H "Authorization: Bearer $API_KEY" \
  -F "file=@logo-dark.png" \
  -F "variant=dark" | jq .
```

Accepted formats: PNG, JPEG, WebP, SVG (max 30MB).

To remove a logo:

```bash title="Delete dark logo"
curl -s -X DELETE "$BASE_URL/api/dashboard/logo/dark" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

## Public Organization Page

Once your profile is set up, your public page at `/o/{slug}` automatically shows:
- Organization name, description, and logo
- All hackathons you've organized
- All hackathons you've sponsored

```bash title="Fetch public organization profile"
curl -s "$BASE_URL/api/public/orgs/agi-ventures" | jq .
```

## Best Practices

1. **Set up your profile early** — Your organization name and logo appear on every hackathon you create
2. **Upload both logo variants** — Light and dark mode support ensures your branding looks good everywhere
3. **Use a descriptive slug** — Keep it short, memorable, and matching your brand (e.g., `agi-ventures` not `agi-ventures-canada-inc`)
4. **Add a website URL** — Visitors to your organization page can click through to learn more
5. **Keep your description current** — The public page is a landing page for potential participants and sponsors
