Set up webhooks to get notified when hackathons are created or updated, and when submissions are created or submitted.

## Overview

Webhooks allow you to:
- Receive real-time notifications for hackathon and submission events
- Avoid polling for status changes
- Build event-driven integrations

## Creating a Webhook

```typescript title="create-webhook.ts"
const res = await fetch(`${BASE_URL}/api/v1/webhooks`, {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    url: "https://your-app.com/webhooks/oatmeal",
    events: ["hackathon.created", "submission.created"],
  }),
})

const webhook = await res.json()

// Save the secret for verifying signatures
console.log(`Webhook secret: ${webhook.secret}`)
```

<Callout type="warn">
The webhook secret is only shown once. Store it securely for signature verification.
</Callout>

## Available Events

| Event | Description |
|-------|-------------|
| `hackathon.created` | A new hackathon was created |
| `hackathon.updated` | A hackathon's settings were updated |
| `participant.registered` | A user registered for a hackathon |
| `submission.created` | A team created a new submission |
| `submission.submitted` | A submission was finalized and submitted |
| `submission.updated` | A team updated their submission |
| `results.published` | Hackathon results were published |

## Webhook Payload

Webhook requests include:
- `event` - The event type
- `timestamp` - ISO 8601 timestamp of when the event occurred
- `data` - Event-specific data including resource IDs and metadata

```json title="payload.json"
{
  "event": "submission.created",
  "timestamp": "2026-02-10T15:30:00Z",
  "data": {
    "hackathonId": "hack_abc123",
    "submissionId": "sub_xyz456",
    "title": "AI Code Review Bot"
  }
}
```

## Verifying Signatures

Webhook requests include an `X-Webhook-Signature` header containing an HMAC-SHA256 hex digest of the request body, signed with your webhook secret.

### Express Example

```typescript title="express-webhook.ts"
import express from "express"
import crypto from "crypto"

const app = express()
app.use(express.json())

const WEBHOOK_SECRET = process.env.WEBHOOK_SECRET!

app.post("/webhooks/oatmeal", (req, res) => {
  const signature = req.headers["x-webhook-signature"] as string
  const payload = JSON.stringify(req.body)
  const expected = crypto
    .createHmac("sha256", WEBHOOK_SECRET)
    .update(payload)
    .digest("hex")

  if (signature !== expected) {
    return res.status(401).send("Invalid signature")
  }

  const { event, data } = req.body

  switch (event) {
    case "hackathon.created":
      console.log(`Hackathon created: ${data.hackathonId}`)
      break
    case "submission.created":
      console.log(`Submission received: ${data.submissionId}`)
      break
  }

  res.status(200).send("OK")
})
```

### Next.js API Route

```typescript title="app/api/webhooks/route.ts"
import { NextRequest, NextResponse } from "next/server"
import crypto from "crypto"

export async function POST(req: NextRequest) {
  const signature = req.headers.get("x-webhook-signature")
  const body = await req.text()

  const expected = crypto
    .createHmac("sha256", process.env.WEBHOOK_SECRET!)
    .update(body)
    .digest("hex")

  if (signature !== expected) {
    return NextResponse.json({ error: "Invalid signature" }, { status: 401 })
  }

  const { event, data } = JSON.parse(body)

  if (event === "submission.created") {
    await processSubmission(data.submissionId, data.title)
  }

  return NextResponse.json({ received: true })
}
```

## Managing Webhooks

### List Webhooks

```typescript title="list-webhooks.ts"
const res = await fetch(`${BASE_URL}/api/v1/webhooks`, {
  headers: {
    "Authorization": `Bearer ${API_KEY}`,
  },
})

const { webhooks } = await res.json()

for (const wh of webhooks) {
  console.log(`${wh.url} - ${wh.events.join(", ")}`)
}
```

### Delete a Webhook

```typescript title="delete-webhook.ts"
const res = await fetch(`${BASE_URL}/api/v1/webhooks/${webhookId}`, {
  method: "DELETE",
  headers: {
    "Authorization": `Bearer ${API_KEY}`,
  },
})

const data = await res.json()
if (data.success) {
  console.log("Webhook deleted")
}
```

## Best Practices

1. **Always verify signatures** - Reject requests with invalid signatures
2. **Respond quickly** - Return 200 within 30 seconds to avoid timeouts
3. **Process asynchronously** - Queue heavy processing for background jobs
4. **Handle retries** - Webhooks may be retried on failure; use idempotency keys
