Process documents through an AI agent and handle results asynchronously.

## Overview

This guide shows how to:
- Start an agent run with document context
- Poll for completion with configurable timeouts
- Handle success and failure cases

## Implementation

<CodeBlock language="typescript" showLineNumbers code={"import { createClient } from \"@agents-server/sdk\"\n\nconst client = createClient(process.env.AGENTS_API_KEY!)\n\nasync function processDocument(documentUrl: string) {\n  // Start the agent run\n  const { data: run, error } = await client.agents.run(\"doc-processor-agent\", {\n    prompt: `Analyze and summarize the document at: ${documentUrl}`,\n    context: { documentUrl }\n  })\n\n  if (error) {\n    throw new Error(`Failed to start agent: ${error.error}`)\n  }\n\n  // Wait for completion (polls every 2 seconds, max 5 minutes)\n  const result = await client.agents.waitForResult(run.runId, {\n    intervalMs: 2000,\n    maxAttempts: 150\n  })\n\n  if (result.status === \"succeeded\") {\n    return result.result\n  } else {\n    throw new Error(`Agent failed: ${JSON.stringify(result.error)}`)\n  }\n}"}>

## Customizing Polling

The `waitForResult` method accepts options to control polling behavior:

| Option | Default | Description |
|--------|---------|-------------|
| `intervalMs` | 1000 | Time between polls in milliseconds |
| `maxAttempts` | 120 | Maximum number of poll attempts |

For long-running documents, increase `maxAttempts`:

<CodeBlock language="typescript" code={"const result = await client.agents.waitForResult(run.runId, {\n  intervalMs: 5000,    // Poll every 5 seconds\n  maxAttempts: 360     // Wait up to 30 minutes\n})"}>

## Error Handling

Always handle both API errors and agent failures:

<CodeBlock language="typescript" showLineNumbers code={"try {\n  const { data: run, error } = await client.agents.run(agentId, { prompt })\n\n  if (error) {\n    // API error (auth, rate limit, etc.)\n    console.error(\"API error:\", error.error)\n    return\n  }\n\n  const result = await client.agents.waitForResult(run.runId)\n\n  if (result.status === \"failed\") {\n    // Agent execution failed\n    console.error(\"Agent error:\", result.error)\n    return\n  }\n\n  // Success\n  console.log(result.result)\n} catch (err) {\n  // Network error or timeout\n  console.error(\"Unexpected error:\", err)\n}"}>
