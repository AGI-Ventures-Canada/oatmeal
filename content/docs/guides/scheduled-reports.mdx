Create schedules to run agents automatically at specified intervals.

## Overview

Schedules let you:
- Run agents on a recurring basis (hourly, daily, weekly, monthly)
- Use cron expressions for custom schedules
- Specify timezone for accurate timing
- Pass input data to each run

## Creating a Schedule

<CodeBlock language="typescript" showLineNumbers code={"import { createClient } from \"@agents-server/sdk\"\n\nconst client = createClient(process.env.AGENTS_API_KEY!)\n\n// Create a daily report schedule at 9 AM Eastern\nconst { data: schedule } = await client.schedules.create({\n  name: \"Daily Sales Report\",\n  frequency: \"daily\",\n  timezone: \"America/New_York\",\n  agentId: \"sales-report-agent\",\n  input: { reportType: \"daily-summary\" }\n})\n\nconsole.log(`Schedule created: ${schedule.id}`)\nconsole.log(`Next run: ${schedule.nextRunAt}`)"}>

## Frequency Options

| Frequency | Description |
|-----------|-------------|
| `once` | Run once at specified time |
| `hourly` | Run every hour |
| `daily` | Run once per day |
| `weekly` | Run once per week |
| `monthly` | Run once per month |
| `cron` | Custom cron expression |

## Using Cron Expressions

For custom schedules, use a cron expression:

<CodeBlock language="typescript" showLineNumbers code={"// Every Monday and Friday at 3:30 PM\nconst { data: schedule } = await client.schedules.create({\n  name: \"Bi-weekly Report\",\n  frequency: \"cron\",\n  cronExpression: \"30 15 * * 1,5\",\n  timezone: \"America/Los_Angeles\",\n  agentId: \"report-agent\",\n  input: { reportType: \"bi-weekly\" }\n})"}>

Common timezones:
- `America/New_York` - US Eastern
- `America/Los_Angeles` - US Pacific
- `Europe/London` - UK
- `Asia/Tokyo` - Japan
- `UTC` - Coordinated Universal Time
