title: Scheduled Reports
description: Automate recurring agent runs with schedules.

Create schedules to run agents automatically at specified intervals.

## Overview

Schedules let you:
- Run agents on a recurring basis (hourly, daily, weekly, monthly)
- Use cron expressions for custom schedules
- Specify timezone for accurate timing
- Pass input data to each run

## Creating a Schedule

```typescript
import { createClient } from "@agents-server/sdk"

const client = createClient(process.env.AGENTS_API_KEY!)

// Create a daily report schedule at 9 AM Eastern
const { data: schedule } = await client.schedules.create({
  name: "Daily Sales Report",
  frequency: "daily",
  timezone: "America/New_York",
  agentId: "sales-report-agent",
  input: { reportType: "daily-summary" }
})

console.log(`Schedule created: ${schedule.id}`)
console.log(`Next run: ${schedule.nextRunAt}`)
```

## Frequency Options

| Frequency | Description |
|-----------|-------------|
| `once` | Run once at specified time |
| `hourly` | Run every hour |
| `daily` | Run once per day |
| `weekly` | Run once per week |
| `monthly` | Run once per month |
| `cron` | Custom cron expression |

## Using Cron Expressions

For custom schedules, use a cron expression:

```typescript
// Every Monday and Friday at 3:30 PM
const { data: schedule } = await client.schedules.create({
  name: "Bi-weekly Report",
  frequency: "cron",
  cronExpression: "30 15 * * 1,5",
  timezone: "America/Los_Angeles",
  agentId: "report-agent",
  input: { reportType: "bi-weekly" }
})
```

## Managing Schedules

### List Active Schedules

```typescript
const { data: schedules } = await client.schedules.list({ activeOnly: true })
console.log(`Active schedules: ${schedules.schedules.length}`)

for (const schedule of schedules.schedules) {
  console.log(`- ${schedule.name}: next run at ${schedule.nextRunAt}`)
}
```

### Pause a Schedule

```typescript
await client.schedules.update(scheduleId, { isActive: false })
```

### Resume a Schedule

```typescript
await client.schedules.update(scheduleId, { isActive: true })
```

### Delete a Schedule

```typescript
await client.schedules.delete(scheduleId)
```

## Timezone Support

Always specify a timezone to ensure runs happen at the expected time:

```typescript
const { data: schedule } = await client.schedules.create({
  name: "Morning Digest",
  frequency: "daily",
  timezone: "Europe/London",  // Runs at local time in London
  agentId: "digest-agent"
})
```

Common timezones:
- `America/New_York` - US Eastern
- `America/Los_Angeles` - US Pacific
- `Europe/London` - UK
- `Asia/Tokyo` - Japan
- `UTC` - Coordinated Universal Time
