---
title: Crowd Voting API
description: Export vote counts and crowd voting results via the REST API.
---

The Crowd Voting API allows attendees and the public to vote for their favorite submissions. Each user gets one vote per hackathon — voting for a new submission replaces the previous vote. As an organizer, use the public endpoint to export vote counts for reporting.

## Endpoints

All endpoints use the public namespace with hackathon slugs.

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/public/hackathons/:slug/vote` | Cast or change vote |
| `DELETE` | `/api/public/hackathons/:slug/vote` | Remove vote |
| `GET` | `/api/public/hackathons/:slug/vote` | Get vote counts |

Casting and removing votes requires a Clerk session. Reading vote counts is public.

## Get Vote Counts

`GET /api/public/hackathons/:slug/vote` — optional Clerk session.

Returns aggregated vote counts for all submissions and the current user's vote (if authenticated).

```bash title="Get vote counts"
curl -s "$BASE_URL/api/public/hackathons/ai-innovation-2026/vote" | jq .
```

### Response

```json title="response.json"
{
  "userVote": "sub_xyz456",
  "counts": [
    { "submissionId": "sub_xyz456", "voteCount": 24 },
    { "submissionId": "sub_abc789", "voteCount": 18 },
    { "submissionId": "sub_def012", "voteCount": 12 }
  ]
}
```

When not authenticated, `userVote` is `null`.

## Cast a Vote

`POST /api/public/hackathons/:slug/vote` — requires Clerk session.

This is a participant action. Attendees cast votes through the hackathon event page. If the user has already voted in this hackathon, their vote moves to the new submission.

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `submissionId` | `string` | Yes | Submission to vote for |

## Remove Vote

`DELETE /api/public/hackathons/:slug/vote` — requires Clerk session.

This is a participant action. Attendees remove their vote through the hackathon event page. Returns an error if the user hasn't voted.

## Full Example

Export vote counts sorted by popularity:

```bash title="Export vote counts"
curl -s "$BASE_URL/api/public/hackathons/ai-innovation-2026/vote" | jq '.counts | sort_by(-.voteCount)'
```
