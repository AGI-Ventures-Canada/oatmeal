---
title: Prizes API
description: Create and manage hackathon prizes, reorder display, and assign winners via the REST API.
---

The Prizes API lets organizers define prize categories, control display order, and assign prizes to winning submissions.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/dashboard/hackathons/:id/prizes` | List prizes and assignments |
| `POST` | `/api/dashboard/hackathons/:id/prizes` | Create a prize |
| `PATCH` | `/api/dashboard/hackathons/:id/prizes/:prizeId` | Update a prize |
| `DELETE` | `/api/dashboard/hackathons/:id/prizes/:prizeId` | Delete a prize |
| `POST` | `/api/dashboard/hackathons/:id/prizes/reorder` | Reorder prizes |
| `POST` | `/api/dashboard/hackathons/:id/prizes/:prizeId/assign` | Assign prize to submission |
| `DELETE` | `/api/dashboard/hackathons/:id/prizes/:prizeId/assign/:submissionId` | Unassign prize |
| `GET` | `/api/public/hackathons/:slug/prizes` | List prizes (public) |

Dashboard endpoints require an API key with `hackathons:read` or `hackathons:write` scope. The public endpoint is unauthenticated.

## Response Types

```typescript title="types.ts"
type PrizeType = "score" | "favorite" | "crowd" | "criteria"

interface Prize {
  id: string
  name: string
  description: string | null
  value: string | null
  type: PrizeType
  rank: number | null
  kind: string
  monetaryValue: number | null
  currency: string
  distributionMethod: string | null
  displayValue: string | null
  criteriaId: string | null
  displayOrder: number
  createdAt: string
}

interface PrizeAssignment {
  id: string
  prizeId: string
  prizeName: string
  submissionId: string
  submissionTitle: string
  teamName: string
  assignedAt: string
}
```

### Prize Types

| Type | Description |
|------|-------------|
| `score` | Awarded based on numeric score rankings |
| `favorite` | Judge pick / editor's choice |
| `crowd` | Crowd voting winner |
| `criteria` | Best in a specific judging criteria |

## List Prizes

`GET /api/dashboard/hackathons/:id/prizes` — requires `hackathons:read` scope.

Returns all prizes and their current assignments.

```bash title="List prizes"
curl -s "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon prizes list $HACKATHON_ID
```
</Callout>

## Create a Prize

`POST /api/dashboard/hackathons/:id/prizes` — requires `hackathons:write` scope.

```bash title="Create a prize"
curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Best Overall",
    "description": "Top scoring project across all criteria",
    "value": "$5,000 USD",
    "type": "score",
    "rank": 1,
    "monetaryValue": 5000,
    "currency": "USD"
  }' | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon prizes create $HACKATHON_ID
```
</Callout>

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | `string` | Yes | Prize name |
| `description` | `string \| null` | No | Prize description |
| `value` | `string \| null` | No | Display value (e.g., "$5,000 USD") |
| `type` | `PrizeType` | No | Prize type (default: `"favorite"`) |
| `rank` | `number \| null` | No | Placement rank (1st, 2nd, 3rd) |
| `kind` | `string` | No | Category kind (default: `"other"`) |
| `monetaryValue` | `number \| null` | No | Numeric monetary value |
| `currency` | `string` | No | Currency code (default: `"USD"`) |
| `distributionMethod` | `string \| null` | No | How prize is distributed |
| `displayValue` | `string \| null` | No | Custom display value |
| `criteriaId` | `string \| null` | No | Linked criteria (for `"criteria"` type) |
| `displayOrder` | `number` | No | Display position |

## Reorder Prizes

`POST /api/dashboard/hackathons/:id/prizes/reorder` — requires `hackathons:write` scope.

Sets display order based on array position.

```bash title="Reorder prizes"
curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes/reorder" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "orderedIds": ["prize_001", "prize_002", "prize_003"]
  }' | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon prizes reorder $HACKATHON_ID
```
</Callout>

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `orderedIds` | `string[]` | Yes | Prize IDs in desired display order |

## List Public Prizes

`GET /api/public/hackathons/:slug/prizes` — no authentication required.

Returns prizes with winner information (if the hackathon is completed and results are published).

```bash title="View public prizes"
curl -s "$BASE_URL/api/public/hackathons/ai-innovation-2026/prizes" | jq .
```

## Full Example

```bash title="Create prizes and set display order"
#!/usr/bin/env bash
set -euo pipefail

PRIZE_1=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Best Overall",
    "value": "$5,000",
    "type": "score",
    "rank": 1
  }' | jq -r '.id')

PRIZE_2=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Runner Up",
    "value": "$2,500",
    "type": "score",
    "rank": 2
  }' | jq -r '.id')

PRIZE_3=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Most Creative",
    "value": "$1,000",
    "type": "favorite"
  }' | jq -r '.id')

PRIZE_4=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "People'\''s Choice",
    "value": "$1,000",
    "type": "crowd"
  }' | jq -r '.id')

curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/prizes/reorder" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d "{
    \"orderedIds\": [\"$PRIZE_1\", \"$PRIZE_2\", \"$PRIZE_3\", \"$PRIZE_4\"]
  }" | jq .

echo "Created and ordered 4 prizes"
```
