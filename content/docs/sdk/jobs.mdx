---
title: Jobs API
description: Create and manage async jobs via the REST API.
---

The Jobs API allows you to queue async tasks and poll for their results.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/v1/jobs` | Create a new job |
| `GET` | `/api/v1/jobs/:id` | Get job status |
| `GET` | `/api/v1/jobs/:id/result` | Get job result |
| `POST` | `/api/v1/jobs/:id/cancel` | Cancel a job |

All endpoints require an API key with `hackathons:read` or `hackathons:write` scope.

## Response Types

```typescript title="types.ts"
type JobStatus = "queued" | "running" | "succeeded" | "failed" | "canceled"

interface Job {
  id: string
  type: string
  status: JobStatus
  createdAt: string
  updatedAt?: string
  completedAt?: string | null
}

interface JobResult {
  id: string
  status: JobStatus
  result: unknown
  error: unknown
  completedAt: string
}
```

## Create a Job

`POST /api/v1/jobs` — requires `hackathons:write` scope.

```bash title="Create a job"
curl -s -X POST "$BASE_URL/api/v1/jobs" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "process-document",
    "input": {
      "url": "https://example.com/document.pdf",
      "options": {"extractTables": true}
    },
    "idempotencyKey": "doc-abc-123"
  }' | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon jobs create --type process-document --input '{"url": "https://example.com/document.pdf"}'
```
</Callout>

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `type` | `string` | Yes | Job type identifier |
| `input` | `object` | No | Job input payload (JSON) |
| `idempotencyKey` | `string` | No | Unique key to prevent duplicate jobs |

The idempotency key can also be sent as an `Idempotency-Key` HTTP header.

### Response

```json title="response.json"
{
  "id": "job_abc123",
  "type": "process-document",
  "status": "queued",
  "createdAt": "2026-02-10T15:30:00Z"
}
```

## Get Job Status

`GET /api/v1/jobs/:id` — requires `hackathons:read` scope.

```bash title="Get job status"
curl -s "$BASE_URL/api/v1/jobs/$JOB_ID" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon jobs get $JOB_ID
```
</Callout>

### Response

```json title="response.json"
{
  "id": "job_abc123",
  "type": "process-document",
  "status": "running",
  "createdAt": "2026-02-10T15:30:00Z",
  "updatedAt": "2026-02-10T15:30:05Z",
  "completedAt": null
}
```

## Get Job Result

`GET /api/v1/jobs/:id/result` — requires `hackathons:read` scope.

Returns `202` with `{ "error": "Job not completed", "status": "running" }` if the job is still in progress.

```bash title="Get job result"
curl -s "$BASE_URL/api/v1/jobs/$JOB_ID/result" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon jobs result $JOB_ID
```
</Callout>

### Response (completed)

```json title="response.json"
{
  "id": "job_abc123",
  "status": "succeeded",
  "result": { "pages": 12, "tables": 3 },
  "error": null,
  "completedAt": "2026-02-10T15:31:00Z"
}
```

## Polling Helper

For convenience, here is a simple polling loop you can use to wait for job completion:

```bash title="Poll for job completion"
while true; do
  STATUS=$(curl -s "$BASE_URL/api/v1/jobs/$JOB_ID" \
    -H "Authorization: Bearer $API_KEY" | jq -r '.status')
  [ "$STATUS" = "succeeded" ] || [ "$STATUS" = "failed" ] && break
  echo "Status: $STATUS — waiting..."
  sleep 2
done

curl -s "$BASE_URL/api/v1/jobs/$JOB_ID/result" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

## Cancel a Job

`POST /api/v1/jobs/:id/cancel` — requires `hackathons:write` scope.

Cancels a queued or running job. Cannot cancel already-completed jobs.

```bash title="Cancel a job"
curl -s -X POST "$BASE_URL/api/v1/jobs/$JOB_ID/cancel" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon jobs cancel $JOB_ID
```
</Callout>

## Full Example

```bash title="Create a job, poll until done, then fetch the result"
#!/usr/bin/env bash
set -euo pipefail

# Create the job
JOB_ID=$(curl -s -X POST "$BASE_URL/api/v1/jobs" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "process-document",
    "input": {"url": "https://example.com/document.pdf"}
  }' | jq -r '.id')

echo "Created job: $JOB_ID"

# Poll until completion
while true; do
  STATUS=$(curl -s "$BASE_URL/api/v1/jobs/$JOB_ID" \
    -H "Authorization: Bearer $API_KEY" | jq -r '.status')
  [ "$STATUS" = "succeeded" ] || [ "$STATUS" = "failed" ] && break
  echo "Status: $STATUS — waiting..."
  sleep 2
done

echo "Final status: $STATUS"

# Fetch the result
curl -s "$BASE_URL/api/v1/jobs/$JOB_ID/result" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

<Callout type="info" title="CLI equivalent">
```bash
hackathon jobs list
```
</Callout>
