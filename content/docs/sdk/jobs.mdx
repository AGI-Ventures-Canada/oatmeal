The Jobs API allows you to queue async tasks and poll for their results.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/v1/jobs` | Create a new job |
| `GET` | `/api/v1/jobs/:id` | Get job status |
| `GET` | `/api/v1/jobs/:id/result` | Get job result |
| `POST` | `/api/v1/jobs/:id/cancel` | Cancel a job |

All endpoints require an API key with `hackathons:read` or `hackathons:write` scope.

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `type` | `string` | Yes | Job type identifier |
| `input` | `object` | No | Job input payload (JSON) |
| `idempotencyKey` | `string` | No | Unique key to prevent duplicate jobs |

The idempotency key can also be sent as an `Idempotency-Key` HTTP header.

## Get Job Result

`GET /api/v1/jobs/:id/result` — requires `hackathons:read` scope.

Returns `202` with `{ "error": "Job not completed", "status": "running" }` if the job is still in progress.

## Polling Helper

For convenience, here is a simple polling function you can use to wait for job completion:

<CodeBlock language="typescript" showLineNumbers code={"async function waitForResult(\n  jobId: string,\n  options?: { maxAttempts?: number; intervalMs?: number }\n) {\n  const { maxAttempts = 60, intervalMs = 1000 } = options ?? {}\n\n  for (let i = 0; i < maxAttempts; i++) {\n    const res = await fetch(`${BASE_URL}/api/v1/jobs/${jobId}/result`, {\n      headers: { \"Authorization\": `Bearer ${API_KEY}` },\n    })\n\n    if (res.status !== 202) {\n      return await res.json()\n    }\n\n    await new Promise((r) => setTimeout(r, intervalMs))\n  }\n\n  throw new Error(`Job ${jobId} did not complete within ${maxAttempts} attempts`)\n}"}>

## Cancel a Job

`POST /api/v1/jobs/:id/cancel` — requires `hackathons:write` scope.

Cancels a queued or running job. Cannot cancel already-completed jobs.

<CodeBlock language="typescript" showLineNumbers code={"const res = await fetch(`${BASE_URL}/api/v1/jobs/${jobId}/cancel`, {\n  method: \"POST\",\n  headers: {\n    \"Authorization\": `Bearer ${API_KEY}`,\n  },\n})\n\nconst data = await res.json()\nif (data.success) {\n  console.log(\"Job canceled\")\n}"}>
