---
title: Agents API
description: Run AI agents with streaming and human-in-the-loop support.
---

The Agents API allows you to run AI agents, stream their progress, and provide human input when needed.

## Stream Run Progress

Get real-time updates using Server-Sent Events:

<CodeBlock language="typescript" showLineNumbers code={"const eventSource = client.agents.streamRun(\"run_xyz789\")\n\neventSource.onmessage = (event) => {\n  const data = JSON.parse(event.data)\n\n  switch (data.event) {\n    case \"status\":\n      console.log(\"Status update:\", data.data)\n      break\n    case \"step\":\n      console.log(\"Step completed:\", data.data)\n      break\n    case \"done\":\n      console.log(\"Agent finished\")\n      eventSource.close()\n      break\n  }\n}\n\neventSource.onerror = (error) => {\n  console.error(\"Stream error:\", error)\n  eventSource.close()\n}"}>

## Human-in-the-Loop

When an agent requires human input, its status becomes `awaiting_input`. Provide the input to continue:

<CodeBlock language="typescript" showLineNumbers code={"// Check if agent needs input\nconst { data: status } = await client.agents.getRun(\"run_xyz789\")\n\nif (status?.status === \"awaiting_input\") {\n  // Provide the required input\n  const { data } = await client.agents.provideInput(\"run_xyz789\", {\n    input: {\n      approved: true,\n      comments: \"Looks good, proceed with the changes\"\n    }\n  })\n\n  if (data?.success) {\n    console.log(\"Input provided, agent will continue\")\n  }\n}"}>
