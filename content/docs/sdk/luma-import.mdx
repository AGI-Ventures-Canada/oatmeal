---
title: Luma Import API
description: Import hackathon details from Luma events via the REST API.
---

The Luma Import API lets you extract event data from Luma and create a hackathon from it in a two-step flow.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/public/import/luma` | Extract Luma event data |
| `POST` | `/api/dashboard/import/luma` | Create hackathon from import |

The extract endpoint is public. The create endpoint requires an API key with `hackathons:write` scope.

## Step 1: Extract Event Data

`POST /api/public/import/luma` — no authentication required.

Extracts event metadata, sponsors, and prizes from a Luma event page.

```typescript title="extract-luma.ts"
const res = await fetch(`${BASE_URL}/api/public/import/luma`, {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    slug: "ai-hackathon-toronto-2026",
  }),
})

const eventData = await res.json()
console.log("Event:", eventData.name)
console.log("Starts:", eventData.startsAt)
console.log("Sponsors:", eventData.sponsors?.length ?? 0)
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `slug` | `string` | Yes | Luma event slug (from the event URL) |

### Response

The response contains the extracted event data which can be reviewed and modified before creating the hackathon.

## Step 2: Create Hackathon from Import

`POST /api/dashboard/import/luma` — requires `hackathons:write` scope.

Creates a new hackathon using the extracted (and optionally modified) event data. This endpoint also downloads and uploads the banner image if an `imageUrl` is provided.

```typescript title="create-from-luma.ts"
const res = await fetch(`${BASE_URL}/api/dashboard/import/luma`, {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    name: eventData.name,
    description: eventData.description,
    startsAt: eventData.startsAt,
    endsAt: eventData.endsAt,
    registrationOpensAt: eventData.registrationOpensAt,
    registrationClosesAt: eventData.registrationClosesAt,
    locationType: "in_person",
    locationName: eventData.locationName,
    imageUrl: eventData.imageUrl,
    rules: eventData.rules,
    sponsors: eventData.sponsors,
    prizes: eventData.prizes,
  }),
})

const { id, name, slug } = await res.json()
console.log(`Created hackathon: ${name} (${slug})`)
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | `string` | Yes | Hackathon name |
| `description` | `string \| null` | No | Description |
| `startsAt` | `string \| null` | No | Event start (ISO 8601) |
| `endsAt` | `string \| null` | No | Event end (ISO 8601) |
| `registrationOpensAt` | `string \| null` | No | Registration opens (ISO 8601) |
| `registrationClosesAt` | `string \| null` | No | Registration closes (ISO 8601) |
| `locationType` | `"in_person" \| "virtual" \| null` | No | Location type |
| `locationName` | `string \| null` | No | Venue name |
| `locationUrl` | `string \| null` | No | Virtual location URL |
| `imageUrl` | `string \| null` | No | Banner image URL (downloaded and uploaded) |
| `rules` | `string \| null` | No | Rules and guidelines |
| `sponsors` | `array` | No | Array of `{ name, tier }` |
| `prizes` | `array` | No | Array of `{ name, description?, value? }` |

### Response

```json title="response.json"
{
  "id": "hack_abc123",
  "name": "AI Hackathon Toronto 2026",
  "slug": "ai-hackathon-toronto-2026"
}
```

## Full Example

```typescript title="full-example.ts"
const API_KEY = process.env.OATMEAL_API_KEY!
const BASE_URL = process.env.OATMEAL_BASE_URL!

async function importFromLuma(lumaSlug: string) {
  const extractRes = await fetch(`${BASE_URL}/api/public/import/luma`, {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({ slug: lumaSlug }),
  })

  if (!extractRes.ok) {
    throw new Error("Failed to extract Luma event")
  }

  const eventData = await extractRes.json()

  console.log(`Importing: ${eventData.name}`)
  console.log(`  Location: ${eventData.locationName}`)
  console.log(`  Sponsors: ${eventData.sponsors?.length ?? 0}`)
  console.log(`  Prizes: ${eventData.prizes?.length ?? 0}`)

  const createRes = await fetch(`${BASE_URL}/api/dashboard/import/luma`, {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify(eventData),
  })

  if (!createRes.ok) {
    const err = await createRes.json()
    throw new Error(`Import failed: ${err.error}`)
  }

  const { id, slug } = await createRes.json()
  console.log(`\nCreated hackathon: ${slug} (${id})`)
}
```
