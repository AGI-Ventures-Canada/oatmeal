---
title: Luma Import API
description: Import hackathon details from Luma events via the REST API.
---

The Luma Import API lets you extract event data from Luma and create a hackathon from it in a two-step flow.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `POST` | `/api/public/import/luma` | Extract Luma event data |
| `POST` | `/api/dashboard/import/luma` | Create hackathon from import |

The extract endpoint is public. The create endpoint requires an API key with `hackathons:write` scope.

## Step 1: Extract Event Data

`POST /api/public/import/luma` — no authentication required.

Extracts event metadata, sponsors, and prizes from a Luma event page.

```bash title="Extract event data from Luma"
curl -s -X POST "$BASE_URL/api/public/import/luma" \
  -H "Content-Type: application/json" \
  -d '{"slug": "ai-hackathon-toronto-2026"}' | jq .
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `slug` | `string` | Yes | Luma event slug (from the event URL) |

### Response

The response contains the extracted event data which can be reviewed and modified before creating the hackathon.

## Step 2: Create Hackathon from Import

`POST /api/dashboard/import/luma` — requires `hackathons:write` scope.

Creates a new hackathon using the extracted (and optionally modified) event data. This endpoint also downloads and uploads the banner image if an `imageUrl` is provided.

```bash title="Create hackathon from extracted Luma data"
curl -s -X POST "$BASE_URL/api/dashboard/import/luma" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "AI Hackathon Toronto 2026",
    "description": "Build the future of AI",
    "startsAt": "2026-06-01T09:00:00Z",
    "endsAt": "2026-06-02T17:00:00Z",
    "locationType": "in_person",
    "locationName": "MaRS Discovery District",
    "imageUrl": "https://images.lumacdn.com/event-covers/abc.jpg",
    "sponsors": [{"name": "Acme Corp", "tier": "gold"}],
    "prizes": [{"name": "Best Overall", "value": "5000"}]
  }' | jq .
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | `string` | Yes | Hackathon name |
| `description` | `string \| null` | No | Description |
| `startsAt` | `string \| null` | No | Event start (ISO 8601) |
| `endsAt` | `string \| null` | No | Event end (ISO 8601) |
| `registrationOpensAt` | `string \| null` | No | Registration opens (ISO 8601) |
| `registrationClosesAt` | `string \| null` | No | Registration closes (ISO 8601) |
| `locationType` | `"in_person" \| "virtual" \| null` | No | Location type |
| `locationName` | `string \| null` | No | Venue name |
| `locationUrl` | `string \| null` | No | Virtual location URL |
| `imageUrl` | `string \| null` | No | Banner image URL (downloaded and uploaded) |
| `rules` | `string \| null` | No | Rules and guidelines |
| `sponsors` | `array` | No | Array of `{ name, tier }` |
| `prizes` | `array` | No | Array of `{ name, description?, value? }` |

### Response

```json title="response.json"
{
  "id": "hack_abc123",
  "name": "AI Hackathon Toronto 2026",
  "slug": "ai-hackathon-toronto-2026"
}
```

## Full Example

```bash title="Extract from Luma and create a hackathon"
#!/usr/bin/env bash
set -euo pipefail

LUMA_SLUG="ai-hackathon-toronto-2026"

# Step 1: Extract event data
EVENT_DATA=$(curl -s -X POST "$BASE_URL/api/public/import/luma" \
  -H "Content-Type: application/json" \
  -d "{\"slug\": \"$LUMA_SLUG\"}")

echo "Extracted: $(echo "$EVENT_DATA" | jq -r '.name')"
echo "  Location: $(echo "$EVENT_DATA" | jq -r '.locationName')"
echo "  Sponsors: $(echo "$EVENT_DATA" | jq '.sponsors | length')"
echo "  Prizes: $(echo "$EVENT_DATA" | jq '.prizes | length')"

# Step 2: Create hackathon from extracted data
RESULT=$(curl -s -X POST "$BASE_URL/api/dashboard/import/luma" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d "$EVENT_DATA")

echo ""
echo "Created hackathon: $(echo "$RESULT" | jq -r '.slug') ($(echo "$RESULT" | jq -r '.id'))"
```
