---
title: Sponsors API
description: Add, manage, and display hackathon sponsors with tiered visibility via the REST API.
---

The Sponsors API lets organizers manage sponsor listings with tiered visibility, logos, and display ordering.

## Endpoints

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/api/dashboard/hackathons/:id/sponsors` | List sponsors |
| `POST` | `/api/dashboard/hackathons/:id/sponsors` | Add a sponsor |
| `PATCH` | `/api/dashboard/hackathons/:id/sponsors/:sponsorId` | Update sponsor |
| `DELETE` | `/api/dashboard/hackathons/:id/sponsors/:sponsorId` | Remove sponsor |
| `PATCH` | `/api/dashboard/hackathons/:id/sponsors/reorder` | Reorder sponsors |
| `POST` | `/api/dashboard/hackathons/:id/sponsors/:sponsorId/logo` | Upload sponsor logo |
| `DELETE` | `/api/dashboard/hackathons/:id/sponsors/:sponsorId/logo` | Delete sponsor logo |

All endpoints require an API key with `hackathons:read` or `hackathons:write` scope.

## Sponsor Tiers

| Tier | Description |
|------|-------------|
| `title` | Title sponsor — most prominent placement |
| `gold` | Gold tier |
| `silver` | Silver tier |
| `bronze` | Bronze tier |
| `partner` | Partner — non-monetary sponsors |

## Response Types

```typescript title="types.ts"
interface Sponsor {
  id: string
  name: string
  logoUrl: string | null
  websiteUrl: string | null
  tier: "gold" | "silver" | "bronze" | "custom" | null
  customTierLabel: string | null
  displayOrder: number
  sponsorTenantId: string | null
  tenant: {
    slug: string
    name: string
    logoUrl: string | null
    logoUrlDark: string | null
  } | null
  createdAt: string
}
```

## List Sponsors

`GET /api/dashboard/hackathons/:id/sponsors` — requires `hackathons:read` scope.

```bash title="List sponsors"
curl -s "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

## Add a Sponsor

`POST /api/dashboard/hackathons/:id/sponsors` — requires `hackathons:write` scope.

```bash title="Add a sponsor"
curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Anthropic",
    "tier": "gold",
    "websiteUrl": "https://anthropic.com"
  }' | jq .
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | `string` | Yes | Sponsor name |
| `tier` | `string` | No | Sponsor tier |
| `websiteUrl` | `string \| null` | No | Sponsor website |
| `logoUrl` | `string \| null` | No | Logo URL (or upload via logo endpoint) |
| `sponsorTenantId` | `string \| null` | No | Link to an Oatmeal organization |
| `displayOrder` | `number` | No | Display position |

## Update a Sponsor

`PATCH /api/dashboard/hackathons/:id/sponsors/:sponsorId` — requires `hackathons:write` scope.

```bash title="Update a sponsor"
curl -s -X PATCH "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors/$SPONSOR_ID" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "tier": "gold"
  }' | jq .
```

## Remove a Sponsor

`DELETE /api/dashboard/hackathons/:id/sponsors/:sponsorId` — requires `hackathons:write` scope.

```bash title="Remove a sponsor"
curl -s -X DELETE "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors/$SPONSOR_ID" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

## Reorder Sponsors

`PATCH /api/dashboard/hackathons/:id/sponsors/reorder` — requires `hackathons:write` scope.

```bash title="Reorder sponsors"
curl -s -X PATCH "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors/reorder" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "sponsorIds": ["sp_001", "sp_002", "sp_003"]
  }' | jq .
```

### Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `sponsorIds` | `string[]` | Yes | Sponsor IDs in desired display order |

## Upload Sponsor Logo

`POST /api/dashboard/hackathons/:id/sponsors/:sponsorId/logo` — requires `hackathons:write` scope.

Accepts PNG, JPEG, WebP, or SVG files up to 5MB. Supports light and dark variants.

```bash title="Upload sponsor logo"
curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors/$SPONSOR_ID/logo" \
  -H "Authorization: Bearer $API_KEY" \
  -F "file=@logo.png" \
  -F "variant=light" | jq .
```

### Request Body (FormData)

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `file` | `File` | Yes | Image file (max 5MB) |
| `variant` | `"light" \| "dark"` | No | Logo variant (default: `"light"`) |

## Delete Sponsor Logo

`DELETE /api/dashboard/hackathons/:id/sponsors/:sponsorId/logo` — requires `hackathons:write` scope.

```bash title="Delete sponsor logo"
curl -s -X DELETE "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors/$SPONSOR_ID/logo?variant=light" \
  -H "Authorization: Bearer $API_KEY" | jq .
```

## Full Example

```bash title="Add sponsors and set display order"
#!/usr/bin/env bash
set -euo pipefail

SP_1=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Anthropic",
    "tier": "gold",
    "websiteUrl": "https://anthropic.com"
  }' | jq -r '.id')

SP_2=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Vercel",
    "tier": "gold",
    "websiteUrl": "https://vercel.com"
  }' | jq -r '.id')

SP_3=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Supabase",
    "tier": "gold",
    "websiteUrl": "https://supabase.com"
  }' | jq -r '.id')

SP_4=$(curl -s -X POST "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Clerk",
    "tier": "silver",
    "websiteUrl": "https://clerk.com"
  }' | jq -r '.id')

curl -s -X PATCH "$BASE_URL/api/dashboard/hackathons/$HACKATHON_ID/sponsors/reorder" \
  -H "Authorization: Bearer $API_KEY" \
  -H "Content-Type: application/json" \
  -d "{
    \"sponsorIds\": [\"$SP_1\", \"$SP_2\", \"$SP_3\", \"$SP_4\"]
  }" | jq .

echo "Added and ordered 4 sponsors"
```
