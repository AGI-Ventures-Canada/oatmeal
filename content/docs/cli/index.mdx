The Oatmeal CLI lets you manage hackathons, judges, prizes, and results directly from your terminal. It works on Mac, Linux, and Windows — no Bun required.

## Installation

Install globally or run directly with `npx`:

```bash title="Run without installing"
npx @agi-ventures-canada/hackathon-cli --version
```

```bash title="Install globally"
npm install -g @agi-ventures-canada/hackathon-cli
```

Or with Bun:

```bash title="Using Bun"
bunx @agi-ventures-canada/hackathon-cli --version
```

## Two Modes

The CLI supports two modes of operation:

| Mode | Auth Required | Commands |
|------|---------------|----------|
| **Browse** | No | Search hackathons, view submissions, check results |
| **Manage** | Yes | Create events, manage judges, assign prizes, publish results |

Browse commands work immediately — no login needed:

```bash title="Public browsing"
hackathon browse hackathons
hackathon browse results my-hackathon
```

Management commands require authentication:

```bash title="Authenticated management"
hackathon login
hackathon events list
hackathon prizes create <hackathon-id> --name "Best AI App"
```

## Command Reference

### Global Flags

| Flag | Description |
|------|-------------|
| `--help` | Show help for any command |
| `--version` | Show CLI version |
| `--json` | Output raw JSON (for scripting) |
| `--yes` | Skip confirmation prompts |
| `--base-url <url>` | Override the API URL |
| `--api-key <key>` | Use a specific API key |

### Auth

| Command | Description |
|---------|-------------|
| `hackathon login` | Sign in via browser (auto-creates API key) |
| `hackathon login --api-key sk_live_...` | Sign in with an existing API key |
| `hackathon login --no-browser` | Paste an API key manually |
| `hackathon logout` | Clear saved credentials |
| `hackathon whoami` | Show current tenant and scopes |
| `hackathon update` | Update CLI to latest version |

### Browse (No Auth)

| Command | Description |
|---------|-------------|
| `hackathon browse hackathons` | Search public hackathons |
| `hackathon browse submissions <slug>` | List submissions |
| `hackathon browse results <slug>` | View published results |
| `hackathon browse org <slug>` | View organization profile |

### Events

| Command | Description |
|---------|-------------|
| `hackathon events list` | List your hackathons |
| `hackathon events get <id>` | Get hackathon details |
| `hackathon events create` | Create a new hackathon |
| `hackathon events update <id>` | Update hackathon settings |
| `hackathon events delete <id>` | Delete a hackathon |

### Judging

| Command | Description |
|---------|-------------|
| `hackathon judging criteria list <id>` | List judging criteria |
| `hackathon judging criteria create <id>` | Add a criterion |

| `hackathon judging judges add <id>` | Add a judge |
| `hackathon judging judges remove <id> <pid>` | Remove a judge |
| `hackathon judging assignments list <id>` | List judge assignments |
| `hackathon judging assignments create <id>` | Create an assignment |
| `hackathon judging assignments delete <id> <aid>` | Delete an assignment |
| `hackathon judging auto-assign <id>` | Auto-distribute submissions |
| `hackathon judging invitations list <id>` | List pending invitations |
| `hackathon judging invitations cancel <id> <iid>` | Cancel an invitation |
| `hackathon judging pick-results <id>` | View subjective judging tallies |
