Welcome to the Oatmeal documentation. This guide covers how to interact with the Oatmeal platform via the CLI and REST API.

## Overview

Oatmeal provides two ways to interact with the platform:

- **[CLI](/docs/cli)** - Manage hackathons, judges, prizes, and results from your terminal
- **[REST API](/docs/getting-started)** - Programmatic access for jobs, webhooks, and schedules
