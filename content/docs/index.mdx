Welcome to the Oatmeal documentation. This guide covers the TypeScript SDK for integrating with the Oatmeal platform API.

## Overview

Oatmeal provides a robust API for:

- **Jobs** - Queue and execute async tasks with status tracking
- **Agents** - Run AI agents with streaming, human-in-the-loop, and result polling
