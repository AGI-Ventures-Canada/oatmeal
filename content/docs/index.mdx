Welcome to the Oatmeal documentation. This guide covers how to interact with the Oatmeal platform via the CLI, REST API, and AI agent skills.

## Overview

Oatmeal provides three ways to interact with the platform:

- **[CLI](/docs/cli)** — Manage hackathons, judges, prizes, and results from your terminal
- **[REST API](/docs/getting-started)** — Full programmatic access to hackathons, submissions, judging, and more
- **[Hackathon Skills](/docs/skills)** — Installable AI agent skills for managing, planning, and competing in hackathons

## Authentication

API endpoints use either an API key or a Clerk session depending on the namespace:

```bash title="API Key (programmatic access)"
Authorization: Bearer sk_live_your_api_key_here
```

See [Getting Started](/docs/getting-started) for details on authentication modes.
