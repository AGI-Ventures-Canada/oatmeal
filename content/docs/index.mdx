Welcome to the Oatmeal API documentation. This guide covers how to integrate with the Oatmeal platform using direct REST API calls.

## Overview

Oatmeal provides a REST API for:

- **Jobs** - Queue and execute async tasks with status tracking and polling
- **Webhooks** - Receive real-time notifications for hackathon and submission events
- **Schedules** - Create recurring jobs with cron expressions and timezone support
