Welcome to the Agents Server documentation. This guide covers the TypeScript SDK for integrating with the Agents Server API.

## Overview

Agents Server provides a robust API for:

- **Jobs** - Queue and execute async tasks with status tracking
- **Agents** - Run AI agents with streaming, human-in-the-loop, and result polling

## Authentication

All API endpoints require authentication via API key:

```bash
Authorization: Bearer sk_live_your_api_key_here
```
