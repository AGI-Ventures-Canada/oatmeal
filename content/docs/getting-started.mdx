The official TypeScript SDK provides a type-safe client for interacting with the Agents Server API.

**Features:**
- Full TypeScript support with exported types
- Promise-based API with error handling
- Built-in polling helpers (`waitForResult`)
- Zero external dependencies (uses native `fetch`)

## Installation

<Tabs items={['npm', 'pnpm', 'bun']} persist groupId="package-manager">
  <Tab value="npm">
  ```bash
  npm install @agents-server/sdk
  ```
  </Tab>
  <Tab value="pnpm">
  ```bash
  pnpm add @agents-server/sdk
  ```
  </Tab>
  <Tab value="bun">

## Create a Client

Initialize the client with your API key:

<CodeBlock language="typescript" code={"import { createClient } from \"@agents-server/sdk\"\n\nconst client = createClient(process.env.AGENTS_API_KEY!)"}>
  <CodeBlockHeader>
    <CodeBlockFilename>lib/agents.ts</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />
    </CodeBlockActions>
  </CodeBlockHeader>
</CodeBlock>

The SDK automatically detects `NEXT_PUBLIC_APP_URL` for local development, or defaults to the production API.

<Callout type="warn">
Never expose your API key in client-side code. Use environment variables and server-side API routes.
</Callout>

## Verify Connection

Test your connection with the `whoami` endpoint:

<CodeBlock language="typescript" showLineNumbers code={"const { data, error } = await client.whoami()\n\nif (error) {\n  console.error(\"Authentication failed:\", error.error)\n  process.exit(1)\n}\n\nconsole.log(\"Connected as tenant:\", data.tenantId)\nconsole.log(\"Key ID:\", data.keyId)\nconsole.log(\"Scopes:\", data.scopes)"}>

## Response Format

All SDK methods return an `ApiResponse<T>` object:

<CodeBlock language="typescript" code={"interface ApiResponse<T> {\n  data: T | null       // Response data on success\n  error: { error: string } | null  // Error message on failure\n  status: number       // HTTP status code\n}"}>
  <CodeBlockHeader>
    <CodeBlockFilename>types.ts</CodeBlockFilename>
    <CodeBlockActions>
      <CodeBlockCopyButton />

  </CodeBlockHeader>

This allows consistent error handling:

<CodeBlock language="typescript" showLineNumbers code={"const { data, error, status } = await client.agents.get(\"agent_123\")\n\nif (error) {\n  console.error(`Error ${status}: ${error.error}`)\n} else {\n  console.log(data)\n}"}>
